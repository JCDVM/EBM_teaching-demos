EBM Teaching Demos

Interactive tools for the p-values and confidence intervals lesson.

1. Sample size & effect size demo
Draw repeated samples to see how n and the true effect drive p-values and CI coverage.
2. Outcome variability demo
See how outcome SD (via SE = SD / sqrt(n)) affects p-values and CI width.
3. Probability of replication
See how an observed p-value relates to the odds an exact replication also hits p<0.05.
4. The winner's curse
Only significant results get counted — watch the average estimate exaggerate the true effect under low power.
